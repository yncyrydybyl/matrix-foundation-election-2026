# Matrix Foundation — 2026 Governing Board Election

A single-page infographic explaining the 2026 Governing Board election of [The Matrix.org Foundation](https://matrix.org/foundation/) — timeline, board composition, voting cycle, the seats on offer, and how to get involved.

🔗 **Live:** https://yncyrydybyl.github.io/matrix-foundation-election-2026/

## What's in here

- `index.html` — the entire infographic, bundled into a single self-contained file (HTML + CSS + JSX + fonts inlined). No build step, no dependencies. Open it in a browser, or host it anywhere static files are served.

## Built-in tweaks

Click the small **"Tweaks"** dot in the bottom-right of the page to open a controls panel:

- **Brand mark** — switch between the official `[matrix]` wordmark and three original geometric variations (lattice / bracket / glyph)
- **Imagery** — toggle the hero band, switch between flowing lines, ASCII density grid, network graph, or data bars
- **Color** — accent hue and saturation
- **Surface** — background grid, scanlines overlay, mono vs. serif titles, density (compact / regular / comfy)

## Deploy your own

This repo is configured for GitHub Pages out of the box.

1. Fork or clone.
2. Push to `main`.
3. **Settings → Pages → Source:** *Deploy from branch* → `main` / `(root)`.
4. Pages will build at `https://<your-username>.github.io/<repo-name>/`.
